# Katastata

Приложение для мониторинга времени проведенного в программах.

Функции программы:
1. Сканирование запущенных программ
2. Сохранение инфомации о сессии: Даты запуска, закрытия, проведенного времени
3. Сохранение общей статистики: Общее время, последний запуск
4. Поддержка нескольких пользователей
5. Разделение программ на категории
